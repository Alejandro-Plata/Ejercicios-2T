Examen UT3 modelo D

Evaluación de JavaScript y Manipulación del DOM

Completa cada ejercicio según sus especificaciones.

La valoración de cada uno de ellos es de 2,5pts.

Ejercicio 1: ⭐ Estrellas Fugaces

En este ejercicio el usuario podrá activar y desactivar el efecto de estrellas fugaces haciendo clic derecho para alternar el seguimiento mientras mueve el ratón lo que generará estrellas en la posición del puntero que desaparecerán automáticamente después de medio segundo.

  • 🖱️ Haz clic derecho una vez para activar la estela y otra vez para detenerla.

    • Ejercicio 2: 📊 Control de Barras

    En este ejercicio el usuario podrá seleccionar y modificar el tamaño de diferentes barras verticales dentro de un contenedor usando la rueda del ratón para cambiar la barra activa haciendo clic izquierdo para aumentar su altura y clic derecho para disminuirla mientras que las barras tienen un tamaño mínimo de 50px y un máximo de 300px. Además, el menú contextual del navegador se deshabilitará para evitar interferencias con el clic derecho

  • 🖱️ Usa la rueda del ratón para seleccionar una barra.
  • 🔼 Usa el Scroll +Ctrl para aumentar la barra.
  • 🔽 Usa el Scroll +Ctrl reducir la barra.

    • Ejercicio 3: 🚗 Usa los Intermitentes

    En esta actividad se simula el funcionamiento de los intermitentes de un coche, donde el usuario puede activarlos y apagarlos mediante el teclado. Al presionar la tecla "←" (izquierda) o "→" (derecha), se encenderá el intermitente correspondiente y comenzará a parpadear (0.5 segundos cada parpadeo). Si el usuario pulsa la tecla opuesta, el intermitente encendido se apagará sin activar el otro.

    🐭 Atrapa el Ratón 🎯

    En esta actividad deberás atrapar un ratón que se mueve aleatoriamente dentro de una ratonera, donde cada segundo el ratón cambiará de posición y, si haces clic sobre él ganarás un punto y se moverá inmediatamente a una nueva ubicación, mientras que si fallas y haces clic en la ratonera aparecerá un mensaje indicando el error, reiniciando la racha. Puedes obtener las posiciones aleatorias dentro de la ratonera con las siguientes funciones...

    • Math.floor(Math.random() * (ratonera.clientHeight - 40))
    • Math.floor(Math.random() * (ratonera.clientWidth - 40))
